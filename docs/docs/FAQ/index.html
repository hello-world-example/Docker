容器启动后闪退

Docker 容器后台运行，就必须有一个前台进程

docker run -d centos /bin/bash -c "while true; do echo noting; sleep 1; done"

root 用户进行容器

docker exec -it --user root <container id> /bin/bash

$ ls -al /var/lib/docker/containers
